# adaxisoft-api

An API that will generate random names

## Usage

```sh
npm install
npm start
```

```/name/``` returns a name

```/name/10``` returns ten names

```/name/first``` returns a first name

```/name/first/6``` returns six first names

```/name/last``` returns a last name

```/name/last/2``` returns two last names
